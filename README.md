Cadastro de Clientes com Validação PHP & Cookies

Um sistema simples e didático de cadastro de clientes que demonstra a integração de uma interface responsiva em HTML, validação de dados no lado do servidor com PHP e persistência temporária de informações utilizando Cookies do navegador.
🚀 Funcionalidades
Formulário HTML Estruturado: Interface amigável para captura de dados do cliente (Nome, E-mail, Telefone, CPF).
Validação Robusta em PHP: Verificação de campos obrigatórios, formato de e-mail e higienização de inputs para evitar vulnerabilidades.
Persistência com Cookies: Armazenamento seguro de preferências ou dados da sessão atual do cliente diretamente no navegador.
Feedback de Erro e Sucesso: Mensagens claras exibidas ao usuário caso a validação falhe ou o cadastro seja bem-sucedido.
📋 Pré-requisitos

Para rodar este projeto localmente, você precisará de um ambiente de servidor local que suporte PHP:PHP >= 8.0
Servidor Web local como XAMPP, WampServer, Laragon, ou o próprio servidor embutido do PHP.
